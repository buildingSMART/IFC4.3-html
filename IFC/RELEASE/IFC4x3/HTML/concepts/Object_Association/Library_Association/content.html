4.2.5 Library Association

The concept Library Association describes how object and object types can have associated library entities indicating the another source of data such as from a model server, product library, or external database or system that enriches the data with more details. Libraries may be referenced in their entirety from projects or project libraries indicating the master source and version of data. Contents within libraries may be referenced from any object, type object, property, and some resource schema entities within a project or project library.
