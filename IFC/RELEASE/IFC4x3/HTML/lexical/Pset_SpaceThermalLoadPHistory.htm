6.2.4.26 Pset_SpaceThermalLoadPHistory

6.2.4.26.1 Semantic definition

The space thermal load IfcSpaceThermalLoadProperties defines actual measured thermal losses and gains occurring within a space or zone. The thermal load source attribute defines an enumeration of possible sources of the thermal load.

6.2.4.26.2 Applicable entities

PSET_PERFORMANCEDRIVEN The property sets defined by this IfcPropertySetTemplate can only be assigned to IfcPerformanceHistory, which is related to the applicable object by means of IfcRelAssignsToControl.
