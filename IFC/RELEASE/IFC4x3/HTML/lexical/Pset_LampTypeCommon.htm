7.4.4.38 Pset_LampTypeCommon

7.4.4.38.1 Semantic definition

A lamp is a component within a light fixture that is designed to emit light.

History: Name changed from Pset_LampEmitterTypeCommon in IFC 2x3.

7.4.4.38.2 Applicable entities

PSET_TYPEDRIVENOVERRIDE The property sets defined by this IfcPropertySetTemplate can be assigned to subtypes of IfcTypeObject and can be overridden by a property set with same name at subtypes of IfcObject.

  • IfcLampType
