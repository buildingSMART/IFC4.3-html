7.5.4.5 Pset_AirTerminalTypeCommon

7.5.4.5.1 Semantic definition

Air terminal type common attributes. SoundLevel attribute deleted in IFC2x2 Pset Addendum: Use IfcSoundProperties instead.

7.5.4.5.2 Applicable entities

PSET_TYPEDRIVENOVERRIDE The property sets defined by this IfcPropertySetTemplate can be assigned to subtypes of IfcTypeObject and can be overridden by a property set with same name at subtypes of IfcObject.
