7.4.4.46 Pset_ProtectiveDeviceBreakerUnitTypeMCB

7.4.4.46.1 Semantic definition

A coherent set of attributes representing the breaking capacities of an MCB. Note - A protective device may be associated with different instances of this property set providing information related to different basic characteristics.

7.4.4.46.2 Applicable entities

PSET_TYPEDRIVENOVERRIDE The property sets defined by this IfcPropertySetTemplate can be assigned to subtypes of IfcTypeObject and can be overridden by a property set with same name at subtypes of IfcObject.
