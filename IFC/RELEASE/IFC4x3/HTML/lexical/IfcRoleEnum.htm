8.1.2.3 IfcRoleEnum

8.1.2.3.1 Semantic definition

This enumeration defines roles which may be played by an actor.

HISTORY New type in IFC1.5.
IFC4-CHANGE Spelling of COMMISSIONINGENGINEER corrected.

8.1.2.3.2 Type values

8.1.2.3.3 Formal representation 

TYPE IfcRoleEnum = ENUMERATION OF
	(ARCHITECT
	,BUILDINGOPERATOR
	,BUILDINGOWNER
	,CIVILENGINEER
	,CLIENT
	,COMMISSIONINGENGINEER
	,CONSTRUCTIONMANAGER
	,CONSULTANT
	,CONTRACTOR
	,COSTENGINEER
	,ELECTRICALENGINEER
	,ENGINEER
	,FACILITIESMANAGER
	,FIELDCONSTRUCTIONMANAGER
	,MANUFACTURER
	,MECHANICALENGINEER
	,OWNER
	,PROJECTMANAGER
	,RESELLER
	,STRUCTURALENGINEER
	,SUBCONTRACTOR
	,SUPPLIER
	,USERDEFINED);
END_TYPE;
