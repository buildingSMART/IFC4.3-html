6.1.2.18 IfcRoofTypeEnum

6.1.2.18.1 Semantic definition

This enumeration defines the basic configuration of the roof in terms of the different roof shapes, as illustrated in Figure 1.

Roofs which are subdivided into more than these basic shapes or roofs with non-regular shapes (free form roofs) have the type FREEFORM.

HISTORY New enumeration in IFC2x.

6.1.2.18.2 Type values

6.1.2.18.3 Formal representation 

TYPE IfcRoofTypeEnum = ENUMERATION OF
	(BARREL_ROOF
	,BUTTERFLY_ROOF
	,DOME_ROOF
	,FLAT_ROOF
	,FREEFORM
	,GABLE_ROOF
	,GAMBREL_ROOF
	,HIPPED_GABLE_ROOF
	,HIP_ROOF
	,MANSARD_ROOF
	,PAVILION_ROOF
	,RAINBOW_ROOF
	,SHED_ROOF
	,USERDEFINED
	,NOTDEFINED);
END_TYPE;
