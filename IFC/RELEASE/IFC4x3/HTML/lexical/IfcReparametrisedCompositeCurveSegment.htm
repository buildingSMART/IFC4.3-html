8.9.3.57 IfcReparametrisedCompositeCurveSegment

8.9.3.57.1 Semantic definition

The IfcReparametrisedCompositeCurveSegment is geometrically identical to a IfcCompositeCurveSegment but with the additional capability of reparametrization.

NOTE Definition according to ISO/CD 10303-42:1992 The reparametrised composite curve segment is a special type of composite curve segment which provides the capability to re-define its parametric length without changing its geometry.
NOTE Entity adapted from reparametrised_composite_curve_segment in ISO 10303-42.
