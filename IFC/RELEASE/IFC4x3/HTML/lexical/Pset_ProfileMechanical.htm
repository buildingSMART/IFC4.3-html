8.15.4.3 Pset_ProfileMechanical

8.15.4.3.1 Semantic definition

This is a collection of mechanical properties that are applicable to virtually all profile classes. Most of these properties are especially used in structural analysis.

8.15.4.3.2 Applicable entities

PSET_PROFILEDRIVEN The property sets defined by this IfcPropertySetTemplate are to be encoded in an IfcProfileProperties entity and assigned to an IfcProfileDef.
