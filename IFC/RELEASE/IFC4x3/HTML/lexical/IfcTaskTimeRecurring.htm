8.5.3.10 IfcTaskTimeRecurring

8.5.3.10.1 Semantic definition

IfcTaskTimeRecurring is a recurring instance of IfcTaskTime for handling regularly scheduled or repetitive tasks.

HISTORY New entity in IFC4.
