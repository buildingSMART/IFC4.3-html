8.9.3.22 IfcCompositeCurveSegment

8.9.3.22.1 Semantic definition

An IfcCompositeCurveSegment is a bounded curve constructed for the sole purpose to be a segment within an IfcCompositeCurve.

NOTE Definition according to ISO/CD 10303-42:1992 A composite curve segment is a bounded curve together with transition information which is used to construct a composite curve.
NOTE Entity adapted from composite_curve_segment defined in ISO 10303-42.
HISTORY New entity in IFC1.0

8.9.3.22.5 Formal representation 

ENTITY IfcCompositeCurveSegment
 SUPERTYPE OF (ONEOF
	(IfcReparametrisedCompositeCurveSegment))
 SUBTYPE OF (IfcSegment);
	SameSense : IfcBoolean;
	ParentCurve : IfcCurve;
 DERIVE
	 Dim : IfcDimensionCount := ParentCurve.Dim;
 WHERE
	ParentIsBoundedCurve : ('IFC4X3.IFCBOUNDEDCURVE' IN TYPEOF(ParentCurve));
END_ENTITY;
