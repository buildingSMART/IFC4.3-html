7.5.2.27 IfcPumpTypeEnum

7.5.2.27.1 Semantic definition

Defines general types of pumps.

HISTORY New enumeration in IFC2x. SUBMERSIBLEPUMP and SUMPPUMP added in IFC4.

7.5.2.27.2 Type values

7.5.2.27.3 Formal representation 

TYPE IfcPumpTypeEnum = ENUMERATION OF
	(CIRCULATOR
	,ENDSUCTION
	,SPLITCASE
	,SUBMERSIBLEPUMP
	,SUMPPUMP
	,VERTICALINLINE
	,VERTICALTURBINE
	,USERDEFINED
	,NOTDEFINED);
END_TYPE;
