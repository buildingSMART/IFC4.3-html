7.4.2.6 IfcCommunicationsApplianceTypeEnum

7.4.2.6.1 Semantic definition

Defines the range of different types of communications appliance that can be specified.

HISTORY New enumeration in IFC4.

7.4.2.6.2 Type values

7.4.2.6.3 Formal representation 

TYPE IfcCommunicationsApplianceTypeEnum = ENUMERATION OF
	(ANTENNA
	,AUTOMATON
	,COMPUTER
	,FAX
	,GATEWAY
	,INTELLIGENTPERIPHERAL
	,IPNETWORKEQUIPMENT
	,LINESIDEELECTRONICUNIT
	,MODEM
	,NETWORKAPPLIANCE
	,NETWORKBRIDGE
	,NETWORKHUB
	,OPTICALLINETERMINAL
	,OPTICALNETWORKUNIT
	,PRINTER
	,RADIOBLOCKCENTER
	,REPEATER
	,ROUTER
	,SCANNER
	,TELECOMMAND
	,TELEPHONYEXCHANGE
	,TRANSITIONCOMPONENT
	,TRANSPONDER
	,TRANSPORTEQUIPMENT
	,USERDEFINED
	,NOTDEFINED);
END_TYPE;
