Example overview

SpatialStructure_2

Intent

This UT establishes a simple spatial structure of a bridge.

It is based on the storyline IR_SLBH-AM.

Prerequisites

This scenario builds upon following other scenarios: - ProjectSetup_1

Content

This scenario covers the following concepts and/or IFC entities: - spatial structure - IfcSite - IfcBridge - IfcFacilityPart with correct PredefinedType
